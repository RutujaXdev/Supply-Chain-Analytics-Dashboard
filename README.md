# Supply Chain Analytics Dashboard

A comprehensive end-to-end Business Intelligence project built using **Python, PostgreSQL, SQL, DAX, and Power BI** to analyze sales, inventory, logistics, and supplier performance.

## Project Overview
## Tech Stack

- Power BI
- PostgreSQL
- SQL
- DAX
- Power Query

---

### Executive Dashboard

- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Monthly Sales Trend
- Sales by Category
- Payment Method Analysis
- Top Products

### Sales Dashboard

- Total Sales
- Total Profit
- Average Order Value
- Total Quantity Sold
- Monthly Sales Trend
- Profit by Category
- Sales by State
- Top Customers

### Inventory Dashboard

- Inventory Value
- Total Quantity
- Total Products
- Warehouse Capacity
- Inventory by Category

- Top Products

### Logistics Dashboard

- Shipping Cost
- Delivered Orders
- Average Delivery Days
- Returned Orders
- Order Status Analysis
- Shipping Cost by State
- Shipping Cost by Payment Method

### Supplier Performance Dashboard

- Total Suppliers
- Average Supplier Rating
- Total Products
- Supplier Sales
- Supplier Profit
- Supplier Performance by Country
